Documentation for Web Development Basics

HTML

HTML (Hypertext Markup Language) is used to structure content on the web.

Basic Structure

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
  </body>
</html>

HTML elements represent the building blocks of any website and include elements such as headings, paragraphs, links, and images.

HTML Paragraph Tag

The paragraph tag in HTML is used to define blocks of text as paragraphs, delineated with <p> tags.

Example

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

Paragraphs automatically have some white space around them, making text easier to read.

Image Tag

The <img> tag is used to embed images in an HTML document.

Attributes

  • src - The path to the image file.
  • alt - Textual description of the image, which is crucial for accessibility and SEO.

Example

<img src="path/to/image.jpg" alt="A description of the image">

CSS

CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation of a document written in HTML.

Basic Structure

selector {
  property: value;
}

Example

body {
  background-color: lightblue;
}

h1 {
  color: navy;
}

By using CSS, you can manipulate the appearance of all elements on your site uniformly and easily.

JavaScript

JavaScript is a high-level, interpreted scripting language that enables you to create dynamically updating content, control multimedia, and animate images.

Basic Example

<script>
  document.getElementById("demo").innerHTML = "Hello JavaScript!";
</script>

Functions and Events

<button onclick="myFunction()">Click me</button>

<script>
  function myFunction() {
    alert('Button clicked!');
  }
</script>

JavaScript enables interactivity on web pages, turning them into more dynamic, user-friendly experiences.
